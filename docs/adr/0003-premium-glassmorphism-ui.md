# ADR-0003: Premium Glassmorphism UI Design System

**Status**: Accepted  
**Date**: 2026-07-01

## Context

前端原先为浅色调、基础 CSS、无设计系统的一致风格。随功能增多（分析、回测、历史、版本），页面风格不统一：

- K 线图和 AgentStream 终端使用深色背景，其他页面使用浅色背景
- 无 CSS 变量体系，颜色、间距、圆角硬编码
- 首页操作区（标的输入、模型选择）布局分散，缺少"自选股"管理
- 分析过程（SSE 流）与结果（图表、报告）混排在同一区块，体验不够聚焦

## Decision

采用统一的设计方案，核心决策点：

### 1. 深空暗黑主题（Dark Glassmorphism）

全站切换为深色背景（`hsl(240, 18%, 6%)`），配合磨砂玻璃卡片效果：

- `backdrop-filter: blur(20px)` 实现毛玻璃质感
- 半透明边框（`hsla(0, 0%, 100%, 0.07)`）模拟玻璃边缘
- 多层级阴影营造空间感

### 2. 三区布局（Variant A）

选择侧栏经典的 Variant A 布局：

- 左侧 260px 固定侧栏 → 自选股管理
- 顶部操作栏 → 模型选择 + 快速操作按钮
- 主内容区 → 行情图表 + 技术分析报告

### 3. 分析弹窗分离

SSE 流式执行过程从内联区块改为居中浮层（AnalysisModal）：

- 用户点击"开始分析"后弹窗展示实时执行进度
- 分析完成后可关闭，结果在主区域呈现
- 分散了"过程"与"结果"的注意力焦点

### 4. CSS 变量设计系统

全局 `:root` 变量统一管理色彩、玻璃质感、文字色：

- 所有组件引用 `var(--xxx)` 而非硬编码色值
- 5 组表面色、5 组强调色、3 级文字色
- 完整的玻璃质感参数（blur、border、shadow）

### 5. 自选股持久化

使用 `localStorage` 存储用户自选股列表，刷新不丢失。
添加自选股通过浮层搜索完成（而非浏览器 prompt）。

## Rationale

- **暗黑主题一致性**：K 线图与 AgentStream 终端已使用深色，全站统一降低视觉割裂
- **玻璃质感**：比纯色卡片更具层次感和科技感，适合金融分析工具的专业调性
- **三区布局**：左侧自选股、顶部操作、中间内容的"交易软件"经典模式，用户认知成本低
- **弹窗分离**：解决原版 SSE 流与结果混排的问题，过程可观察但不干扰结果阅读
- **CSS 变量**：为后续主题切换和组件复用打好基础

## Consequences

- 旧页面（HistoryView、HistoryDetailView 等）尚未适配暗黑主题，后续需逐步迁移
- 自选股搜索浮层目前基于本地已知标的列表，未来可对接后端搜索 API
- CSS 变量体系引入后，新增组件必须引用 `var(--xxx)`，不得硬编码色值
- 需要维护 `docs/agents/CONTEXT.md` 设计系统文档，与代码同步更新
