## 分析框架

### 1. 行情概览

- 最新价格和涨跌幅
- 均线系统（MA5/10/20/60）的多空排列状态
- 常用指标解读：MACD（趋势和动量）、RSI（超买超卖）、KDJ（短线方向）、BOLL（支撑阻力位和带宽）、ATR（波动率）
- 成交量分析

### 2. 技术形态识别

- 当前K线形态（如锤子线、吞没形态等）
- 波浪理论或道氏理论的市场阶段判断
- 重要支撑和阻力位

### 3. 五维交叉验证评分

对以下五个维度进行打分（+1/0/-1）并给出依据：

1. **趋势维度**：均线排列、MA60 位置、趋势线的方向
2. **动量维度**：MACD 金叉/死叉、RSI 位置和方向、动量指标
3. **波动维度**：ATR 绝对值和相对位置、BOLL 带宽变化
4. **量价维度**：成交量与价格的关系、量价背离
5. **位置风险**：价格在 BOLL 中的位置、前高前低、历史分位数

### 4. 综合结论

- 综合评分汇总
- 多空判断（强烈看多/看多/中性/看空/强烈看空）
- 关键价位（支撑位和阻力位列表）
- 风险提示（至少3条）
